# Task 28: Quick Start Guide - Receptionist Check-in/out Interface

## เริ่มต้นใช้งาน

### 1. เข้าสู่ระบบ

1. เปิดเบราว์เซอร์และไปที่ `http://localhost:3000`
2. เข้าสู่ระบบด้วยบัญชี Receptionist
3. ระบบจะพาไปที่แดชบอร์ดห้องพัก

### 2. เมนูหลัก

หลังจากเข้าสู่ระบบ คุณจะเห็นเมนูด้านบน:
- **แดชบอร์ดห้องพัก** - ดูสถานะห้องทั้งหมด
- **เช็คอิน** - ทำการเช็คอินแขก
- **เช็คเอาท์** - ทำการเช็คเอาท์แขก
- **ย้ายห้อง** - ย้ายแขกไปห้องอื่น
- **No-Show** - จัดการแขกที่ไม่มา

## การใช้งานแต่ละหน้า

### หน้าเช็คอิน

**ขั้นตอน:**

1. **เลือกวันที่**
   - ระบบจะแสดงวันนี้เป็นค่าเริ่มต้น
   - สามารถเปลี่ยนวันที่ได้

2. **เลือกแขก**
   - คลิกที่การ์ดแขกจากรายการด้านซ้าย
   - ดูรายละเอียด: ชื่อ, ประเภทห้อง, วันที่เข้าพัก

3. **เลือกห้อง**
   - ระบบจะแสดงห้องว่างที่พร้อมใช้งาน
   - ห้องที่มีป้าย "ตรวจสอบแล้ว" (สีเขียว) พร้อมใช้งานที่สุด
   - ห้องที่มีป้าย "สะอาด" (สีเหลือง) ก็ใช้ได้

4. **ยืนยันเช็คอิน**
   - กดปุ่ม "ยืนยันเช็คอิน"
   - รอสักครู่
   - ระบบจะแจ้งเตือนเมื่อสำเร็จ

**ตัวอย่าง:**
```
แขก: นาย สมชาย ใจดี
ประเภทห้อง: Deluxe Room
เช็คอิน: 15 ม.ค. 2567
เช็คเอาท์: 17 ม.ค. 2567

→ เลือกห้อง 201 (ตรวจสอบแล้ว)
→ กดยืนยันเช็คอิน
→ สำเร็จ! แขกได้รับห้อง 201
```

### หน้าเช็คเอาท์

**ขั้นตอน:**

1. **เลือกวันที่**
   - ระบบจะแสดงวันนี้เป็นค่าเริ่มต้น

2. **เลือกแขก**
   - คลิกที่การ์ดแขกจากรายการ
   - ดูรายละเอียด: ชื่อ, หมายเลขห้อง, ยอดรวม

3. **ตรวจสอบรายละเอียด**
   - ชื่อแขก
   - หมายเลขห้อง
   - วันที่เช็คเอาท์
   - ยอดรวมทั้งหมด

4. **ยืนยันเช็คเอาท์**
   - กดปุ่ม "ดำเนินการเช็คเอาท์"
   - อ่านคำเตือน
   - กดปุ่ม "ยืนยันเช็คเอาท์"
   - ระบบจะแจ้งเตือนเมื่อสำเร็จ

**หมายเหตุ:**
- ห้องจะถูกทำเครื่องหมายเป็น "รอทำความสะอาด" อัตโนมัติ
- แผนกแม่บ้านจะได้รับการแจ้งเตือน

**ตัวอย่าง:**
```
แขก: นาย สมชาย ใจดี
ห้อง: 201
เช็คเอาท์: 17 ม.ค. 2567
ยอดรวม: ฿3,000

→ กดดำเนินการเช็คเอาท์
→ อ่านคำเตือน
→ กดยืนยันเช็คเอาท์
→ สำเร็จ! ห้อง 201 ว่างแล้ว (รอทำความสะอาด)
```

### หน้าย้ายห้อง

**ขั้นตอน:**

1. **เลือกแขก**
   - คลิกที่การ์ดแขกที่ต้องการย้ายห้อง
   - ดูห้องปัจจุบัน

2. **เลือกห้องใหม่**
   - ระบบจะแสดงห้องว่างประเภทเดียวกัน
   - เลือกห้องที่ต้องการย้ายไป

3. **กรอกเหตุผล (ไม่บังคับ)**
   - เช่น "แอร์เสีย", "น้ำไม่ไหล", "ขอเปลี่ยนห้อง"

4. **ยืนยันการย้าย**
   - กดปุ่ม "ดำเนินการย้ายห้อง"
   - อ่านคำเตือน
   - กดปุ่ม "ยืนยันย้ายห้อง"
   - ระบบจะแจ้งเตือนเมื่อสำเร็จ

**หมายเหตุ:**
- ห้องเก่าจะถูกทำเครื่องหมายเป็น "รอทำความสะอาด" อัตโนมัติ
- แขกจะย้ายไปห้องใหม่ทันที
- แผนกแม่บ้านจะได้รับการแจ้งเตือน

**ตัวอย่าง:**
```
แขก: นาย สมชาย ใจดี
ห้องปัจจุบัน: 201
ประเภทห้อง: Deluxe Room

→ เลือกห้องใหม่: 203
→ กรอกเหตุผล: "แอร์เสีย"
→ กดดำเนินการย้ายห้อง
→ อ่านคำเตือน
→ กดยืนยันย้ายห้อง
→ สำเร็จ! แขกย้ายไปห้อง 203 แล้ว
```

### หน้า No-Show

**ขั้นตอน:**

1. **ดูรายการแขกที่เกินกำหนด**
   - ระบบจะแสดงการจองที่ยืนยันแล้วแต่แขกยังไม่มา
   - แสดงจำนวนวันที่เกินกำหนด

2. **เลือกการจอง**
   - คลิกที่การ์ดการจองที่ต้องการทำเครื่องหมาย
   - ดูรายละเอียดและข้อมูลติดต่อ

3. **ติดต่อแขก (แนะนำ)**
   - โทรหาแขกก่อนทำเครื่องหมาย No-Show
   - ตรวจสอบว่าแขกอาจมาสายหรือมีปัญหา

4. **ทำเครื่องหมาย No-Show**
   - กดปุ่ม "ทำเครื่องหมาย No-Show"
   - อ่านคำเตือนและคำแนะนำ
   - กดปุ่ม "ยืนยัน No-Show"
   - ระบบจะแจ้งเตือนเมื่อสำเร็จ

**หมายเหตุ:**
- ควรติดต่อแขกก่อนทำเครื่องหมาย No-Show
- การจองจะถูกเปลี่ยนสถานะเป็น "No-Show"
- ห้องจะไม่ถูกปล่อยอัตโนมัติ (ต้องให้ผู้จัดการตัดสินใจ)
- อาจมีการเรียกเก็บค่าปรับตามนโยบาย

**ตัวอย่าง:**
```
แขก: นาย สมชาย ใจดี
ประเภทห้อง: Deluxe Room
วันเช็คอิน (ที่พลาด): 15 ม.ค. 2567
เกิน: 2 วัน
ยอดรวม: ฿3,000
เบอร์โทร: 081-234-5678

→ โทรหาแขก 081-234-5678
→ ไม่สามารถติดต่อได้
→ กดทำเครื่องหมาย No-Show
→ อ่านคำเตือน
→ กดยืนยัน No-Show
→ สำเร็จ! การจองถูกทำเครื่องหมาย No-Show
```

## คำแนะนำการใช้งาน

### เช็คอิน
✅ **ควรทำ:**
- ตรวจสอบเอกสารแขกก่อนเช็คอิน
- เลือกห้องที่มีป้าย "ตรวจสอบแล้ว" ก่อน
- แจ้งหมายเลขห้องและข้อมูลสำคัญให้แขก

❌ **ไม่ควรทำ:**
- เช็คอินก่อนเวลา check-in ที่กำหนด (ถ้าไม่มีการอนุมัติ)
- เลือกห้องที่ยังไม่สะอาด

### เช็คเอาท์
✅ **ควรทำ:**
- ตรวจสอบห้องก่อนเช็คเอาท์ (ถ้าเป็นไปได้)
- แจ้งแผนกแม่บ้านให้ทำความสะอาดห้อง
- ออกใบเสร็จให้แขก

❌ **ไม่ควรทำ:**
- เช็คเอาท์โดยไม่ตรวจสอบยอดรวม
- ลืมแจ้งแผนกแม่บ้าน

### ย้ายห้อง
✅ **ควรทำ:**
- กรอกเหตุผลในการย้ายห้อง
- แจ้งแขกเกี่ยวกับการย้ายห้อง
- ตรวจสอบว่าห้องใหม่พร้อมใช้งาน

❌ **ไม่ควรทำ:**
- ย้ายห้องโดยไม่แจ้งแขก
- ย้ายไปห้องที่ยังไม่สะอาด

### No-Show
✅ **ควรทำ:**
- ติดต่อแขกก่อนทำเครื่องหมาย No-Show
- บันทึกเหตุผล (ถ้ามี)
- แจ้งผู้จัดการเกี่ยวกับ No-Show

❌ **ไม่ควรทำ:**
- ทำเครื่องหมาย No-Show ทันทีโดยไม่ติดต่อแขก
- ลืมแจ้งผู้จัดการ

## การแก้ปัญหา

### ปัญหา: ไม่เห็นรายการแขก
**วิธีแก้:**
1. ตรวจสอบวันที่ที่เลือก
2. รีเฟรชหน้า
3. ตรวจสอบว่ามีการจองในวันนั้นจริง

### ปัญหา: ไม่มีห้องว่าง
**วิธีแก้:**
1. ตรวจสอบสถานะห้องในแดชบอร์ด
2. รอให้แผนกแม่บ้านทำความสะอาดห้อง
3. ติดต่อผู้จัดการเพื่อปล่อยห้องเพิ่ม

### ปัญหา: เช็คอินไม่สำเร็จ
**วิธีแก้:**
1. ตรวจสอบว่าห้องว่างและสะอาด
2. ตรวจสอบว่าการจองมีสถานะ "Confirmed"
3. ลองรีเฟรชหน้าและทำใหม่
4. ติดต่อ IT support ถ้ายังไม่ได้

### ปัญหา: ข้อมูลไม่อัปเดต
**วิธีแก้:**
1. กดปุ่ม "รีเฟรชตอนนี้"
2. รอ 30 วินาที (ระบบจะรีเฟรชอัตโนมัติ)
3. ลองปิดและเปิดหน้าใหม่
