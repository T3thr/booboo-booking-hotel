## Color Scheme

### Dashboard Metric Cards
- **Revenue Card:** Blue gradient (from-blue-50 to-blue-100)
- **Occupancy Card:** Green gradient (from-green-50 to-green-100)
- **ADR Card:** Purple gradient (from-purple-50 to-purple-100)
- **RevPAR Card:** Orange gradient (from-orange-50 to-orange-100)

### Occupancy Rate Badges
- **High (≥80%):** Green background, green text
- **Medium (50-79%):** Yellow background, yellow text
- **Low (<50%):** Red background, red text

### Trend Indicators
- **Positive:** Green text with ↑ arrow
- **Negative:** Red text with ↓ arrow

## Tips for Best Viewing

1. **Use appropriate view mode:**
   - Daily: ≤30 days
   - Weekly: 1-3 months
   - Monthly: 3+ months

2. **Enable comparison for insights:**
   - Compare same periods
   - Identify trends
   - Track improvements

3. **Export for analysis:**
   - Regular backups
   - Excel analysis
   - Stakeholder reports

4. **Monitor key metrics:**
   - Revenue trends
   - Occupancy patterns
   - Pricing effectiveness
