## การเริ่มต้นใช้งาน

### ภาพรวมหน้าที่

ในฐานะแม่บ้าน คุณมีหน้าที่:
- 🧹 ทำความสะอาดห้อง
- 📱 อัปเดตสถานะห้องแบบเรียลไทม์
- 🔧 รายงานปัญหาการซ่อมบำรุง
- ✅ ตรวจสอบห้อง (สำหรับหัวหน้าแม่บ้าน)

### การเข้าสู่ระบบ

**ขั้นตอน:**
1. เปิดเว็บไซต์ `http://localhost:3000`
2. เข้าสู่ระบบด้วยบัญชีแม่บ้าน
3. เลือกบทบาท **"Housekeeper"**
4. คุณจะเข้าสู่หน้ารายการงาน

## รายการงานทำความสะอาด

### ภาพรวมรายการงาน

ระบบจะแสดงห้องที่ต้องทำความสะอาดเรียงตามลำดับความสำคัญ

**📸 ภาพประกอบ: รายการงาน**
```
┌─────────────────────────────────────────────────────────┐
│  🧹 รายการงานทำความสะอาด          🔄 อัปเดต: 10:30   │
│                                                         │
│  สรุป: 🔴 รอทำ: 8  🧽 กำลังทำ: 3  ✅ เสร็จแล้ว: 12  │
│                                                         │
│  กรอง: [ทั้งหมด ▼]  [รอทำ]  [กำลังทำ]  [เสร็จแล้ว]  │
├─────────────────────────────────────────────────────────┤
│  ห้อง  │ ประเภท    │ สถานะ      │ ลำดับ │ การดำเนินการ │
├─────────────────────────────────────────────────────────┤
│  102   │ Deluxe    │ 🔴 รอทำ    │ 🔥 ด่วน │ [เริ่มทำ]  │
│        │           │ (Check-out) │        │            │
├─────────────────────────────────────────────────────────┤
│  105   │ Suite     │ 🔴 รอทำ    │ 🔥 ด่วน │ [เริ่มทำ]  │
│        │           │ (Check-out) │        │            │
├─────────────────────────────────────────────────────────┤
│  108   │ Standard  │ 🔴 รอทำ    │ ⚠️ สูง  │ [เริ่มทำ]  │
│        │           │ (Occupied)  │        │            │
├─────────────────────────────────────────────────────────┤
│  112   │ Deluxe    │ 🧽 กำลังทำ │ ⚠️ สูง  │ [เสร็จแล้ว]│
│        │           │ (คุณทดสอบ)  │        │            │
├─────────────────────────────────────────────────────────┤
│  201   │ Suite     │ 🔴 รอทำ    │ 📅 ปกติ │ [เริ่มทำ]  │
│        │           │ (Vacant)    │        │            │
└─────────────────────────────────────────────────────────┘
```

### ลำดับความสำคัญ

| ลำดับ | ไอคอน | ความหมาย | เหตุผล |
|-------|-------|----------|--------|
| ด่วนที่สุด | 🔥 | Check-out วันนี้ | มีแขกใหม่รอเข้าพัก |
| สูง | ⚠️ | ห้องมีผู้พัก | ต้องทำทุกวัน |
| ปกติ | 📅 | ห้องว่าง | ทำตามลำดับ |
| ต่ำ | ℹ️ | ห้องสะอาดแล้ว | ตรวจสอบเป็นครั้งคราว |

### การเลือกงาน

**แนะนำ:**
1. ทำห้อง 🔥 ด่วนที่สุด ก่อน
2. ทำห้อง ⚠️ สูง ต่อ
3. ทำห้อง 📅 ปกติ สุดท้าย

**💡 เคล็ดลับ:**
- เริ่มจากห้องที่ใกล้กันเพื่อประหยัดเวลา
- ทำห้องเล็กก่อนห้องใหญ่
- แจ้งหัวหน้าถ้างานเยอะเกินไป

## การอัปเดตสถานะห้อง

### วงจรสถานะห้อง

```
🔴 รอทำ (Dirty)
    ↓ [เริ่มทำ]
🧽 กำลังทำ (Cleaning)
    ↓ [เสร็จแล้ว]
🧹 สะอาด (Clean)
    ↓ [ตรวจสอบโดยหัวหน้า]
✅ ตรวจแล้ว (Inspected)
```

### ขั้นตอนการทำความสะอาด

**ขั้นตอนที่ 1: เริ่มทำความสะอาด**
1. เลือกห้องจากรายการ
2. คลิกปุ่ม **"เริ่มทำ"**
3. ระบบจะเปลี่ยนสถานะเป็น **"กำลังทำ"**

**📸 ภาพประกอบ: เริ่มทำความสะอาด**
```
┌─────────────────────────────────────────────┐
│  🧽 เริ่มทำความสะอาด                       │
│                                             │
│  ห้อง: 102 (Deluxe Room)                   │
│  สถานะ: 🔴 รอทำ → 🧽 กำลังทำ             │
│                                             │
│  เวลาโดยประมาณ: 30 นาที                    │
│                                             │
│  ✓ เตรียมอุปกรณ์                           │
│  ✓ เคาะประตูก่อนเข้า                       │
│  ✓ ตรวจสอบของแขก                           │
│                                             │
│  [  ยืนยัน  ]  [  ยกเลิก  ]               │
└─────────────────────────────────────────────┘
```


**ขั้นตอนที่ 2: ทำความสะอาด**
ทำตามขั้นตอนมาตรฐาน:

**สำหรับห้องว่าง (Vacant):**
1. ✓ เปลี่ยนผ้าปูที่นอนและปลอกหมอน
2. ✓ ทำความสะอาดห้องน้ำ
3. ✓ ดูดฝุ่นและถูพื้น
4. ✓ เช็ดฝุ่นเฟอร์นิเจอร์
5. ✓ เติมอุปกรณ์ (สบู่, แชมพู, ผ้าเช็ดตัว)
6. ✓ ตรวจสอบอุปกรณ์ทำงานปกติ
7. ✓ จัดเรียงของให้เป็นระเบียบ

**สำหรับห้องมีผู้พัก (Occupied):**
1. ✓ เคาะประตูและแจ้งตัว
2. ✓ เปลี่ยนผ้าเช็ดตัว (ถ้าแขกต้องการ)
3. ✓ ทำความสะอาดห้องน้ำ
4. ✓ ดูดฝุ่นและถูพื้น
5. ✓ เติมอุปกรณ์
6. ✓ เก็บขยะ
7. ✓ จัดเตียง (ถ้าแขกต้องการ)

**⚠️ ข้อควรระวัง:**
- อย่าเคลื่อนย้ายของแขก
- อย่าเปิดลิ้นชักหรือตู้เสื้อผ้า
- รายงานของมีค่าที่พบทันที

**ขั้นตอนที่ 3: เสร็จสิ้นการทำความสะอาด**
1. ตรวจสอบความสะอาดอีกครั้ง
2. คลิกปุ่ม **"เสร็จแล้ว"** ในระบบ
3. ระบบจะเปลี่ยนสถานะเป็น **"สะอาด"**

**📸 ภาพประกอบ: เสร็จสิ้น**
```
┌─────────────────────────────────────────────┐
│  ✅ ทำความสะอาดเสร็จสิ้น                   │
│                                             │
│  ห้อง: 102 (Deluxe Room)                   │
│  สถานะ: 🧽 กำลังทำ → 🧹 สะอาด            │
│                                             │
│  เวลาที่ใช้: 28 นาที                        │
│  เริ่ม: 10:00  เสร็จ: 10:28                │
│                                             │
│  Checklist:                                │
│  ✓ เปลี่ยนผ้าปู                            │
│  ✓ ทำความสะอาดห้องน้ำ                      │
│  ✓ ดูดฝุ่นและถูพื้น                        │
│  ✓ เติมอุปกรณ์                             │
│  ✓ ตรวจสอบอุปกรณ์                          │
│                                             │
│  [  ยืนยัน  ]                              │
└─────────────────────────────────────────────┘
```

### การบันทึกเวลา

ระบบจะบันทึก:
- เวลาเริ่มทำ
- เวลาเสร็จ
- ระยะเวลาที่ใช้
- ผู้ทำความสะอาด

**💡 ประโยชน์:**
- ติดตามประสิทธิภาพ
- วางแผนงาน
- ปรับปรุงกระบวนการ

## การรายงานปัญหา

### ประเภทปัญหา

| ประเภท | ตัวอย่าง | ความเร่งด่วน |
|--------|----------|--------------|
| 🔧 ซ่อมบำรุง | แอร์เสีย, ก๊อกน้ำรั่ว | สูง |
| 🛏️ อุปกรณ์ | ผ้าปูขาด, หมอนแข็ง | ปานกลาง |
| 🧴 เครื่องใช้ | สบู่หมด, แชมพูหมด | ต่ำ |
| 🚪 โครงสร้าง | ประตูชำรุด, กระจกแตก | สูงมาก |

### ขั้นตอนการรายงาน

**ขั้นตอนที่ 1: เลือกห้องที่มีปัญหา**
1. ในหน้ารายการงาน
2. คลิกปุ่ม **"รายงานปัญหา"** ที่ห้องนั้น

**ขั้นตอนที่ 2: กรอกรายละเอียด**
1. เลือกประเภทปัญหา
2. อธิบายปัญหาอย่างละเอียด
3. เลือกความเร่งด่วน
4. แนบรูปภาพ (ถ้ามี)

**ขั้นตอนที่ 3: ส่งรายงาน**
1. คลิก **"ส่งรายงาน"**
2. ระบบจะเปลี่ยนสถานะห้องเป็น **"ต้องซ่อมบำรุง"**
3. แจ้งเตือนไปยังฝ่ายซ่อมบำรุง

**✅ รายงานสำเร็จ**

│  ✓ แจ้งพนักงานต้อนรับแล้ว                  │
│                                             │

│                                             │

### การติดตามปัญหา

**ดูสถานะการซ่อม:**
1. ไปที่เมนู **"ปัญหาที่รายงาน"**
2. ดูรายการปัญหาทั้งหมด
3. ตรวจสอบสถานะ

**สถานะการซ่อม:**
- ⏳ **รอดำเนินการ**: ยังไม่เริ่มซ่อม
- 🔧 **กำลังซ่อม**: ช่างกำลังซ่อม
- ✅ **เสร็จสิ้น**: ซ่อมเสร็จแล้ว
- ❌ **ยกเลิก**: ไม่ต้องซ่อม

## การตรวจสอบห้อง

### สำหรับหัวหน้าแม่บ้าน

หัวหน้าแม่บ้านมีหน้าที่ตรวจสอบห้องที่ทำความสะอาดเสร็จแล้ว

### ขั้นตอนการตรวจสอบ

**ขั้นตอนที่ 1: ดูรายการห้องรอตรวจ**
1. ไปที่เมนู **"ตรวจสอบห้อง"**
2. ระบบจะแสดงห้องที่มีสถานะ **"สะอาด"**

**📸 ภาพประกอบ: รายการรอตรวจ**
```
┌─────────────────────────────────────────────────────────┐
│  ✅ ตรวจสอบห้อง                                        │
│                                                         │
│  ห้องรอตรวจสอบ: 5 ห้อง                                │
├─────────────────────────────────────────────────────────┤
│  ห้อง  │ ประเภท    │ ทำโดย      │ เวลา   │ การดำเนินการ│
├─────────────────────────────────────────────────────────┤
│  102   │ Deluxe    │ คุณทดสอบ   │ 10:28  │ [ตรวจสอบ]  │
│  105   │ Suite     │ คุณผู้ใช้   │ 10:45  │ [ตรวจสอบ]  │
│  108   │ Standard  │ คุณทดสอบ   │ 11:02  │ [ตรวจสอบ]  │
└─────────────────────────────────────────────────────────┘
```

**ขั้นตอนที่ 2: ตรวจสอบห้อง**
1. คลิกปุ่ม **"ตรวจสอบ"**
2. เข้าไปตรวจห้องจริง
3. ตรวจสอบตาม Checklist

**📸 ภาพประกอบ: Checklist การตรวจ**
```
┌─────────────────────────────────────────────┐
│  ตรวจสอบห้อง 102                           │
│                                             │
│  Checklist:                                │
│  ☐ ผ้าปูเรียบร้อย ไม่มีรอยเปื้อน           │
│  ☐ ห้องน้ำสะอาด ไม่มีคราบ                  │
│  ☐ พื้นสะอาด ไม่มีฝุ่น                     │
│  ☐ เฟอร์นิเจอร์เรียบร้อย                   │
│  ☐ อุปกรณ์ครบ (สบู่, แชมพู, ผ้า)           │
│  ☐ อุปกรณ์ทำงานปกติ (แอร์, ทีวี, ไฟ)       │
│  ☐ ไม่มีกลิ่นไม่พึงประสงค์                 │
│  ☐ ห้องเป็นระเบียบ                         │
│                                             │
│  หมายเหตุ: [_________________]             │
│                                             │
│  [  อนุมัติ  ]  [  ปฏิเสธ  ]              │
└─────────────────────────────────────────────┘
```


**ขั้นตอนที่ 3: อนุมัติหรือปฏิเสธ**

**ถ้าผ่าน:**
1. เลือก Checklist ทั้งหมด
2. คลิก **"อนุมัติ"**
3. สถานะจะเปลี่ยนเป็น **"ตรวจแล้ว"**
4. ห้องพร้อมขาย

**ถ้าไม่ผ่าน:**
1. กรอกหมายเหตุว่าต้องแก้ไขอะไร
2. คลิก **"ปฏิเสธ"**
3. สถานะจะกลับเป็น **"รอทำ"**
4. แจ้งเตือนแม่บ้านที่รับผิดชอบ

**✅ อนุมัติสำเร็จ**
```
┌─────────────────────────────────────────────┐
│  ✅ อนุมัติห้อง 102 สำเร็จ!                │
│                                             │
│  สถานะ: 🧹 สะอาด → ✅ ตรวจแล้ว            │
│                                             │
│  ห้องนี้พร้อมสำหรับ Check-in               │
│  ระบบได้แจ้งพนักงานต้อนรับแล้ว             │
│                                             │
│  [  ตรวจห้องต่อไป  ]  [  เสร็จสิ้น  ]     │
└─────────────────────────────────────────────┘
```

**❌ ปฏิเสธ**
```
┌─────────────────────────────────────────────┐
│  ❌ ปฏิเสธห้อง 102                         │
│                                             │
│  เหตุผล:                                   │
│  • ห้องน้ำยังมีคราบ                        │
│  • พื้นยังมีฝุ่น                            │
│                                             │
│  สถานะ: 🧹 สะอาด → 🔴 รอทำ               │
│                                             │
│  ระบบได้แจ้งคุณทดสอบให้ทำใหม่แล้ว          │
│                                             │
│  [  เสร็จสิ้น  ]                           │
└─────────────────────────────────────────────┘
```

### มาตรฐานการตรวจสอบ

**ห้องน้ำ:**
- ✓ ห้องส้วม สะอาด ไม่มีคราบ
- ✓ อ่างล้างหน้า สะอาด ไม่มีคราบสบู่
- ✓ ฝักบัว สะอาด น้ำไหลปกติ
- ✓ กระจก สะอาด ไม่มีรอยเปื้อน
- ✓ พื้น แห้ง ไม่ลื่น

**ห้องนอน:**

## คำถามที่พบบ่อย (FAQ)

### การทำความสะอาด

**Q: ห้องมีผู้พักแต่ไม่อยู่ ทำอย่างไร?**  
A: เคาะประตู 3 ครั้ง รอ 10 วินาที ถ้าไม่มีคนตอบให้เข้าทำความสะอาด

**Q: แขกไม่ต้องการให้ทำความสะอาด?**  
A: เคารพความต้องการของแขก อัปเดตสถานะเป็น "ไม่รบกวน" ในระบบ

**Q: พบของมีค่าในห้อง?**  
A: อย่าสัมผัส รายงานหัวหน้าทันที บันทึกรายละเอียด

**Q: ทำความสะอาดใช้เวลานานกว่าปกติ?**  
A: อัปเดตสถานะในระบบ แจ้งหัวหน้าถ้าเกิน 1 ชั่วโมง

### ระบบ

**Q: ลืมอัปเดตสถานะ?**  
A: อัปเดตทันทีที่นึกได้ แจ้งหัวหน้าถ้าผ่านไปนาน

**Q: อัปเดตสถานะผิด?**  
A: แจ้งหัวหน้าทันที เพื่อแก้ไขในระบบ

**Q: ระบบช้าหรือค้าง?**  
A: รีเฟรชหน้าเว็บ หรือแจ้งฝ่าย IT

### ปัญหาทั่วไป

**Q: อุปกรณ์ทำความสะอาดหมด?**  
A: แจ้งหัวหน้าเพื่อเติมสต็อก

**Q: ไม่แน่ใจว่าห้องสะอาดพอหรือไม่?**  
A: ให้หัวหน้าตรวจก่อน หรือทำตาม Checklist อย่างเคร่งครัด

**Q: พบปัญหาที่ไม่สามารถแก้ไขได้?**  
A: รายงานปัญหาในระบบทันที อย่าพยายามแก้ไขเอง

### การทำงานอย่างมีประสิทธิภาพ

5. **อัปเดตทันที**: อัปเดตสถานะทุกครั้งที่เสร็จ

### การประหยัดเวลา
