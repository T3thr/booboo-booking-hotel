## การเริ่มต้นใช้งาน

### ภาพรวมหน้าที่

ในฐานะผู้จัดการโรงแรม คุณมีหน้าที่:
- 📊 ดูรายงานและวิเคราะห์ข้อมูล
- 💰 จัดการราคาและระดับราคา
- 📦 จัดการสต็อกห้องพัก
- 📋 จัดการนโยบายและคูปอง
- 🏨 จัดการข้อมูลหลัก (ห้อง, สิ่งอำนวยความสะดวก)
- ⚙️ ตั้งค่าระบบ

### การเข้าสู่ระบบ

**ขั้นตอน:**
1. เปิดเว็บไซต์ `http://localhost:3000`
2. เข้าสู่ระบบด้วยบัญชีผู้จัดการ
3. เลือกบทบาท **"Manager"**
4. คุณจะเข้าสู่แดชบอร์ดผู้จัดการ

### เมนูหลัก

**📊 รายงาน:**
- รายงานการเข้าพัก (Occupancy Report)
- รายงานรายได้ (Revenue Report)
- รายงานคูปอง (Voucher Report)
- รายงาน No-show

**💰 การจัดการราคา:**
- ระดับราคา (Rate Tiers)
- ปฏิทินราคา (Pricing Calendar)
- เมทริกซ์ราคา (Rate Pricing Matrix)

**📦 การจัดการสต็อก:**
- สต็อกห้องพัก (Room Inventory)
- ปฏิทินสต็อก (Inventory Calendar)

**📋 นโยบายและคูปอง:**
- นโยบายการยกเลิก (Cancellation Policies)
- คูปองส่วนลด (Vouchers)

**🏨 ข้อมูลหลัก:**
- ประเภทห้อง (Room Types)
- ห้องพัก (Rooms)
- สิ่งอำนวยความสะดวก (Amenities)

## การจัดการราคา

### 1. การจัดการระดับราคา (Rate Tiers)

ระดับราคาใช้สำหรับแบ่งช่วงเวลาตามอุปสงค์

**ตัวอย่างระดับราคา:**
- **Low Season** (ฤดูต่ำ): เมษายน-กันยายน
- **High Season** (ฤดูสูง): ตุลาคม-มีนาคม
- **Peak Season** (ฤดูสูงสุด): ธันวาคม, เทศกาล
- **Special** (พิเศษ): วันหยุดยาว, อีเวนต์

#### การสร้างระดับราคา

**ขั้นตอน:**
1. ไปที่เมนู **"การจัดการราคา" > "ระดับราคา"**
2. คลิกปุ่ม **"สร้างระดับใหม่"**
3. กรอกข้อมูล
4. บันทึก

**📸 ภาพประกอบ: สร้างระดับราคา**
