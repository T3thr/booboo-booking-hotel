## การเริ่มต้นใช้งาน

### ภาพรวม
ระบบจองโรงแรมช่วยให้คุณสามารถ:
- ค้นหาและจองห้องพักออนไลน์
- ดูประวัติการจองทั้งหมด
- ยกเลิกการจองตามนโยบาย
- ใช้คูปองส่วนลดพิเศษ

### ข้อกำหนด
- อีเมลที่ใช้งานได้
- เบอร์โทรศัพท์
- บัตรเครดิต/เดบิตสำหรับชำระเงิน

## การลงทะเบียนและเข้าสู่ระบบ

### การลงทะเบียนบัญชีใหม่

**ขั้นตอนที่ 1: เข้าสู่หน้าลงทะเบียน**
1. เปิดเว็บไซต์ `http://localhost:3000`
2. คลิกปุ่ม **"ลงทะเบียน"** หรือ **"Sign Up"**

**ขั้นตอนที่ 2: กรอกข้อมูล**
กรอกข้อมูลต่อไปนี้:
- **ชื่อ** (First Name): ชื่อจริงของคุณ
- **นามสกุล** (Last Name): นามสกุลของคุณ
- **อีเมล** (Email): อีเมลที่ใช้งานได้ (จะใช้สำหรับเข้าสู่ระบบ)
- **เบอร์โทรศัพท์** (Phone): เบอร์ติดต่อ
- **รหัสผ่าน** (Password): รหัสผ่านที่มีความปลอดภัย (ขั้นต่ำ 8 ตัวอักษร)
- **ยืนยันรหัสผ่าน** (Confirm Password): กรอกรหัสผ่านอีกครั้ง

**ขั้นตอนที่ 3: ยืนยันการลงทะเบียน**
1. ตรวจสอบข้อมูลให้ถูกต้อง
2. คลิกปุ่ม **"ลงทะเบียน"**
3. ระบบจะส่งอีเมลยืนยันไปที่อีเมลของคุณ


**📸 ภาพประกอบ: หน้าลงทะเบียน**
```
┌─────────────────────────────────────┐
│  🏨 Hotel Booking System           │
│                                     │
│  ลงทะเบียนบัญชีใหม่                │
│                                     │
│  ชื่อ:        [____________]        │
│  นามสกุล:     [____________]        │
│  อีเมล:       [____________]        │
│  เบอร์โทร:    [____________]        │
│  รหัสผ่าน:    [____________]        │
│  ยืนยัน:      [____________]        │
│                                     │
│  [ ลงทะเบียน ]  [ ยกเลิก ]         │
│                                     │
│  มีบัญชีแล้ว? เข้าสู่ระบบ          │
└─────────────────────────────────────┘
```

### การเข้าสู่ระบบ

**ขั้นตอนที่ 1: เข้าสู่หน้า Login**
1. คลิกปุ่ม **"เข้าสู่ระบบ"** หรือ **"Sign In"**

**ขั้นตอนที่ 2: กรอกข้อมูล**
- **อีเมล**: อีเมลที่ลงทะเบียนไว้
- **รหัสผ่าน**: รหัสผ่านของคุณ

**ขั้นตอนที่ 3: เข้าสู่ระบบ**
1. คลิกปุ่ม **"เข้าสู่ระบบ"**
2. ระบบจะพาคุณไปยังหน้าค้นหาห้องพัก

**💡 เคล็ดลับ:**
- เลือก "จดจำฉัน" เพื่อไม่ต้องเข้าสู่ระบบทุกครั้ง
- หากลืมรหัสผ่าน คลิก "ลืมรหัสผ่าน?" เพื่อรีเซ็ต

## การค้นหาห้องพัก

### ขั้นตอนการค้นหา

**ขั้นตอนที่ 1: กรอกข้อมูลการค้นหา**
1. เลือก **วันเช็คอิน** (Check-in Date)
2. เลือก **วันเช็คเอาท์** (Check-out Date)
3. เลือก **จำนวนผู้เข้าพัก** (Number of Guests)
4. คลิกปุ่ม **"ค้นหา"** หรือ **"Search"**

**📸 ภาพประกอบ: ฟอร์มค้นหา**
```
┌─────────────────────────────────────────────┐
│  ค้นหาห้องพัก                              │
│                                             │
│  วันเช็คอิน:    [📅 01/12/2025]            │
│  วันเช็คเอาท์:  [📅 03/12/2025]            │
│  จำนวนผู้พัก:   [👤 2 คน ▼]               │
│                                             │
│  [      🔍 ค้นหาห้องว่าง      ]           │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 2: ดูผลการค้นหา**
ระบบจะแสดงห้องที่ว่างพร้อมข้อมูล:
- **ชื่อประเภทห้อง** (Room Type Name)
- **รูปภาพห้อง**
- **คำอธิบาย** (Description)
- **จำนวนผู้เข้าพักสูงสุด** (Max Occupancy)
- **สิ่งอำนวยความสะดวก** (Amenities)
- **ราคาต่อคืน** (Price per Night)
- **ราคารวม** (Total Price)

**📸 ภาพประกอบ: ผลการค้นหา**
```
┌─────────────────────────────────────────────┐
│  ห้องที่พบ (3 ห้อง)                        │
├─────────────────────────────────────────────┤
│  [รูปภาพ]  Deluxe Room                     │
│            👤 สูงสุด 2 คน                   │
│            ✓ WiFi ✓ TV ✓ แอร์              │
│            💰 2,500 บาท/คืน                │
│            รวม: 5,000 บาท (2 คืน)          │
│            [  จองห้องนี้  ]                │
├─────────────────────────────────────────────┤
│  [รูปภาพ]  Suite Room                      │
│            👤 สูงสุด 4 คน                   │
│            ✓ WiFi ✓ TV ✓ แอร์ ✓ ระเบียง   │
│            💰 4,000 บาท/คืน                │
│            รวม: 8,000 บาท (2 คืน)          │
│            [  จองห้องนี้  ]                │
└─────────────────────────────────────────────┘
```


### เคล็ดลับการค้นหา

✅ **ควรทำ:**
- ค้นหาล่วงหน้าอย่างน้อย 1-2 สัปดาห์
- เปรียบเทียบราคาในวันธรรมดาและวันหยุด
- ตรวจสอบสิ่งอำนวยความสะดวกที่ต้องการ

❌ **ไม่ควรทำ:**
- เลือกวันเช็คเอาท์ก่อนวันเช็คอิน
- เลือกจำนวนผู้พักเกินกว่าที่ห้องรองรับ

## การจองห้องพัก

### ขั้นตอนการจอง

**ขั้นตอนที่ 1: เลือกห้อง**
1. จากผลการค้นหา คลิกปุ่ม **"จองห้องนี้"**
2. ระบบจะสำรองห้องไว้ให้คุณ **15 นาที**

**⏱️ ตัวนับเวลาถอยหลัง**
```
┌─────────────────────────────────────┐
│  ⏰ เวลาที่เหลือ: 14:32            │
│  กรุณาทำการจองให้เสร็จภายในเวลา   │
└─────────────────────────────────────┘
```

**ขั้นตอนที่ 2: กรอกข้อมูลผู้เข้าพัก**
กรอกข้อมูลผู้เข้าพักทุกคน:
- **ชื่อ-นามสกุล** (Full Name)
- **ประเภท**: ผู้ใหญ่ (Adult) หรือ เด็ก (Child)
- **ผู้เข้าพักหลัก**: เลือก 1 คน

**📸 ภาพประกอบ: ฟอร์มข้อมูลผู้พัก**
```
┌─────────────────────────────────────────────┐
│  ข้อมูลผู้เข้าพัก                          │
│                                             │
│  ผู้เข้าพักคนที่ 1 (หลัก)                  │
│  ชื่อ:      [____________]                  │
│  นามสกุล:   [____________]                  │
│  ประเภท:    (•) ผู้ใหญ่  ( ) เด็ก          │
│                                             │
│  ผู้เข้าพักคนที่ 2                         │
│  ชื่อ:      [____________]                  │
│  นามสกุล:   [____________]                  │
│  ประเภท:    (•) ผู้ใหญ่  ( ) เด็ก          │
│                                             │
│  [  ถัดไป  ]                               │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 3: ตรวจสอบและชำระเงิน**
1. ตรวจสอบรายละเอียดการจอง:
   - วันที่เข้าพัก
   - ประเภทห้อง
   - ผู้เข้าพัก
   - ราคารวม
   - นโยบายการยกเลิก

2. ใส่รหัสคูปอง (ถ้ามี)
3. เลือกวิธีชำระเงิน
4. กรอกข้อมูลบัตร
5. คลิก **"ยืนยันการจอง"**

**📸 ภาพประกอบ: หน้าสรุปการจอง**
```
┌─────────────────────────────────────────────┐
│  สรุปการจอง                                │
│                                             │
│  📅 วันที่: 01/12/2025 - 03/12/2025        │
│  🏨 ห้อง: Deluxe Room                      │
│  👥 ผู้พัก: นาย ทดสอบ ระบบหนึ่ง, นาง ผู้ใช้ │
│                                             │
│  💰 ราคา:                                  │
│     ค่าห้อง (2 คืน)      5,000 บาท        │
│     ส่วนลด (SAVE10)       -500 บาท         │
│     ─────────────────────────────          │
│     รวมทั้งสิ้น          4,500 บาท         │
│                                             │
│  📋 นโยบายการยกเลิก:                       │
│     ยกเลิกก่อน 7 วัน: คืนเงิน 100%        │
│     ยกเลิกก่อน 3 วัน: คืนเงิน 50%         │
│     ยกเลิกน้อยกว่า 3 วัน: ไม่คืนเงิน      │
│                                             │
│  รหัสคูปอง: [__________] [ใช้]             │
│                                             │
│  [  ยืนยันและชำระเงิน  ]                  │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 4: รับการยืนยัน**
1. ระบบจะแสดงหน้ายืนยันการจอง
2. คุณจะได้รับอีเมลยืนยันพร้อม:
   - หมายเลขการจอง (Booking ID)
   - รายละเอียดการจอง
   - ใบเสร็จรับเงิน
   - นโยบายการยกเลิก

**✅ การจองสำเร็จ!**
```
┌─────────────────────────────────────────────┐
│  ✅ การจองสำเร็จ!                          │
│                                             │
│  หมายเลขการจอง: #BK-2025-001234           │
│                                             │
│  📧 เราได้ส่งอีเมลยืนยันไปที่:            │
│     your.email@example.com                 │
│                                             │
│  [  ดูรายละเอียด  ]  [  กลับหน้าหลัก  ]   │
└─────────────────────────────────────────────┘
```

## การดูประวัติการจอง

### เข้าสู่หน้าประวัติการจอง

**ขั้นตอน:**
1. เข้าสู่ระบบ
2. คลิกเมนู **"การจองของฉัน"** หรือ **"My Bookings"**
3. ระบบจะแสดงรายการการจองทั้งหมด


**📸 ภาพประกอบ: หน้าประวัติการจอง**
```
┌─────────────────────────────────────────────────────────┐
│  การจองของฉัน                                          │
│                                                         │
│  กรอง: [ทั้งหมด ▼]  [กำลังจะมาถึง]  [เสร็จสิ้น]      │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  #BK-2025-001234  ✅ ยืนยันแล้ว                        │
│  📅 01/12/2025 - 03/12/2025                            │
│  🏨 Deluxe Room                                        │
│  💰 4,500 บาท                                          │
│  [  ดูรายละเอียด  ]  [  ยกเลิก  ]                     │
├─────────────────────────────────────────────────────────┤
│  #BK-2025-001100  ⏳ รอชำระเงิน                        │
│  📅 15/12/2025 - 17/12/2025                            │
│  🏨 Suite Room                                         │
│  💰 8,000 บาท                                          │
│  [  ชำระเงิน  ]  [  ยกเลิก  ]                         │
├─────────────────────────────────────────────────────────┤
│  #BK-2025-000950  ✔️ เสร็จสิ้น                         │
│  📅 20/11/2025 - 22/11/2025                            │
│  🏨 Standard Room                                      │
│  💰 3,000 บาท                                          │
│  [  ดูรายละเอียด  ]  [  ดาวน์โหลดใบเสร็จ  ]          │
└─────────────────────────────────────────────────────────┘
```

### สถานะการจอง

| สถานะ | ความหมาย | การดำเนินการ |
|-------|----------|--------------|
| ⏳ รอชำระเงิน | ยังไม่ได้ชำระเงิน | ชำระเงินหรือยกเลิก |
| ✅ ยืนยันแล้ว | ชำระเงินเรียบร้อย | ดูรายละเอียด, ยกเลิก |
| 🏨 เช็คอินแล้ว | กำลังเข้าพัก | ดูรายละเอียด |
| ✔️ เสร็จสิ้น | เช็คเอาท์แล้ว | ดูรายละเอียด, ดาวน์โหลดใบเสร็จ |
| ❌ ยกเลิกแล้ว | ยกเลิกการจอง | ดูรายละเอียด |
| 🚫 No-show | ไม่มาเช็คอิน | ดูรายละเอียด |

### ดูรายละเอียดการจอง

**ขั้นตอน:**
1. คลิกปุ่ม **"ดูรายละเอียด"** ที่การจองที่ต้องการ
2. ระบบจะแสดงข้อมูลครบถ้วน:
   - หมายเลขการจอง
   - วันที่เข้าพัก
   - ประเภทห้อง
   - หมายเลขห้อง (ถ้าเช็คอินแล้ว)
   - รายชื่อผู้เข้าพัก
   - ราคารายคืน
   - ราคารวม
   - นโยบายการยกเลิก
   - ประวัติการเปลี่ยนแปลง

## การยกเลิกการจอง

### เงื่อนไขการยกเลิก

⚠️ **สำคัญ:** การคืนเงินขึ้นอยู่กับนโยบายการยกเลิกที่บันทึกไว้ตอนจอง

**ตัวอย่างนโยบาย:**
- **ยกเลิกก่อน 7 วัน:** คืนเงิน 100%
- **ยกเลิกก่อน 3 วัน:** คืนเงิน 50%
- **ยกเลิกน้อยกว่า 3 วัน:** ไม่คืนเงิน

### ขั้นตอนการยกเลิก

**ขั้นตอนที่ 1: เลือกการจองที่ต้องการยกเลิก**
1. ไปที่หน้า **"การจองของฉัน"**
2. เลือกการจองที่มีสถานะ **"ยืนยันแล้ว"**
3. คลิกปุ่ม **"ยกเลิก"**

**ขั้นตอนที่ 2: ตรวจสอบเงินคืน**
ระบบจะแสดง:
- จำนวนเงินที่จะได้รับคืน
- เหตุผลการคำนวณ
- ระยะเวลาการคืนเงิน (7-14 วันทำการ)

**📸 ภาพประกอบ: หน้ายืนยันการยกเลิก**
```
┌─────────────────────────────────────────────┐
│  ⚠️ ยืนยันการยกเลิก                        │
│                                             │
│  หมายเลขการจอง: #BK-2025-001234           │
│  วันที่เข้าพัก: 01/12/2025                 │
│                                             │
│  💰 การคืนเงิน:                            │
│     ยอดชำระ:        4,500 บาท              │
│     เงินคืน (100%): 4,500 บาท              │
│                                             │
│  📋 เหตุผล:                                │
│     ยกเลิกก่อนวันเช็คอิน 10 วัน           │
│     ตามนโยบาย: คืนเงิน 100%                │
│                                             │
│  ⏱️ ระยะเวลาคืนเงิน: 7-14 วันทำการ       │
│                                             │
│  [  ยืนยันการยกเลิก  ]  [  ย้อนกลับ  ]    │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 3: ยืนยันการยกเลิก**
1. อ่านข้อมูลให้ครบถ้วน
2. คลิกปุ่ม **"ยืนยันการยกเลิก"**
3. ระบบจะส่งอีเมลยืนยันการยกเลิก

**✅ ยกเลิกสำเร็จ**
- คุณจะได้รับอีเมลยืนยัน
- เงินจะคืนภายใน 7-14 วันทำการ
- สถานะการจองจะเปลี่ยนเป็น **"ยกเลิกแล้ว"**

### ข้อจำกัดการยกเลิก

❌ **ไม่สามารถยกเลิกได้:**
- การจองที่มีสถานะ **"เช็คอินแล้ว"**
- การจองที่มีสถานะ **"เสร็จสิ้น"**
- การจองที่ยกเลิกไปแล้ว

## การใช้คูปองส่วนลด

### วิธีใช้คูปอง

**ขั้นตอน:**
1. ในหน้าสรุปการจอง
2. กรอกรหัสคูปองในช่อง **"รหัสคูปอง"**
3. คลิกปุ่ม **"ใช้"** หรือ **"Apply"**
4. ระบบจะคำนวณส่วนลดอัตโนมัติ

**📸 ภาพประกอบ: การใช้คูปอง**
```
┌─────────────────────────────────────────────┐
│  💰 ราคา:                                  │
│     ค่าห้อง (2 คืน)      5,000 บาท        │
│                                             │
│  รหัสคูปอง: [SAVE10____] [ใช้]             │
│                                             │
│  ✅ ใช้คูปอง SAVE10 สำเร็จ!                │
│     ส่วนลด 10%            -500 บาท         │
│     ─────────────────────────────          │
│     รวมทั้งสิ้น          4,500 บาท         │
└─────────────────────────────────────────────┘
```


### ประเภทคูปอง

| ประเภท | ตัวอย่าง | คำอธิบาย |
|--------|----------|----------|
| เปอร์เซ็นต์ | SAVE10 | ลด 10% จากราคารวม |
| จำนวนเงิน | DISCOUNT500 | ลด 500 บาท |
| ฤดูกาล | SUMMER2025 | ส่วนลดพิเศษช่วงฤดูร้อน |

### ข้อควรระวัง

⚠️ **คูปองอาจไม่สามารถใช้ได้เมื่อ:**
- รหัสไม่ถูกต้อง
- หมดอายุแล้ว
- ถูกใช้ครบจำนวนแล้ว
- ไม่ตรงตามเงื่อนไข (เช่น ใช้ได้เฉพาะห้องบางประเภท)

## คำถามที่พบบ่อย (FAQ)

### การจอง

**Q: ต้องชำระเงินทันทีหรือไม่?**  
A: ใช่ ต้องชำระเงินเต็มจำนวนเพื่อยืนยันการจอง

**Q: สามารถจองหลายห้องพร้อมกันได้หรือไม่?**  
A: ได้ คุณสามารถเพิ่มห้องในการจองเดียวกัน

**Q: ถ้าไม่ชำระเงินภายใน 15 นาทีจะเกิดอะไรขึ้น?**  
A: การจองจะถูกยกเลิกอัตโนมัติและห้องจะกลับมาว่างให้คนอื่นจองได้

**Q: สามารถแก้ไขวันที่หลังจองแล้วได้หรือไม่?**  
A: ไม่ได้ ต้องยกเลิกการจองเดิมและจองใหม่

### การชำระเงิน

**Q: รับชำระด้วยวิธีใดบ้าง?**  
A: รับชำระด้วยบัตรเครดิต/เดบิต (Visa, Mastercard, JCB)

**Q: ข้อมูลบัตรปลอดภัยหรือไม่?**  
A: ปลอดภัย ระบบเข้ารหัสข้อมูลด้วยมาตรฐาน SSL

**Q: จะได้ใบเสร็จหรือไม่?**  
A: ได้ ระบบจะส่งใบเสร็จทางอีเมลทันที

### การยกเลิก

**Q: ยกเลิกแล้วได้เงินคืนเมื่อไร?**  
A: ภายใน 7-14 วันทำการ ขึ้นอยู่กับธนาคาร

**Q: สามารถยกเลิกบางคืนได้หรือไม่?**  
A: ไม่ได้ ต้องยกเลิกทั้งการจอง

**Q: ถ้ายกเลิกช้าเกินไปจะเกิดอะไรขึ้น?**  
A: จะได้เงินคืนน้อยลงหรือไม่ได้เลย ตามนโยบาย

### บัญชีผู้ใช้

**Q: ลืมรหัสผ่านทำอย่างไร?**  
A: คลิก "ลืมรหัสผ่าน?" ในหน้า Login แล้วทำตามขั้นตอน

**Q: สามารถเปลี่ยนอีเมลได้หรือไม่?**  
A: ได้ ในหน้าโปรไฟล์ของคุณ

**Q: ต้องมีบัญชีถึงจะจองได้หรือไม่?**  
A: ใช่ ต้องลงทะเบียนก่อนทำการจอง

### ปัญหาทั่วไป

**Q: ไม่ได้รับอีเมลยืนยัน?**  
A: ตรวจสอบโฟลเดอร์ Spam หรือติดต่อฝ่ายสนับสนุน

**Q: ไม่พบห้องว่างในวันที่ต้องการ?**  
A: ลองเลื่อนวันที่ ±3 วัน หรือเลือกประเภทห้องอื่น

**Q: ระบบช้าหรือค้าง?**  
A: รีเฟรชหน้าเว็บ หรือลองใช้เบราว์เซอร์อื่น

## เคล็ดลับการใช้งาน

### 💡 เคล็ดลับสำหรับการจอง

1. **จองล่วงหน้า**: จองก่อน 1-2 สัปดาห์เพื่อได้ห้องที่ต้องการ
2. **เปรียบเทียบราคา**: ตรวจสอบราคาในวันธรรมดาและวันหยุด
3. **ใช้คูปอง**: ติดตามโปรโมชั่นและคูปองส่วนลด
4. **อ่านนโยบาย**: อ่านนโยบายการยกเลิกก่อนจอง
5. **บันทึกหมายเลข**: เก็บหมายเลขการจองไว้สำหรับอ้างอิง

### 🔒 ความปลอดภัย

1. **รหัสผ่านที่แข็งแกร่ง**: ใช้รหัสผ่านที่มีตัวอักษร ตัวเลข และสัญลักษณ์
2. **ออกจากระบบ**: ออกจากระบบทุกครั้งเมื่อใช้งานเสร็จ
3. **อย่าแชร์บัญชี**: อย่าให้ผู้อื่นใช้บัญชีของคุณ
4. **ตรวจสอบอีเมล**: ตรวจสอบอีเมลยืนยันทุกครั้ง

### 📱 การใช้งานบนมือถือ

1. **เพิ่มไปหน้าจอหลัก**: เพิ่มเว็บไซต์ไปหน้าจอหลักเพื่อเข้าถึงง่าย
2. **ใช้ WiFi**: ใช้ WiFi เมื่อทำการจองเพื่อความเสถียร
3. **หมุนหน้าจอ**: หมุนเป็นแนวนอนเพื่อดูรายละเอียดได้ชัดเจน
