## แดชบอร์ดสถานะห้อง

### ภาพรวมแดชบอร์ด

แดชบอร์ดแสดงสถานะห้องทั้งหมดแบบเรียลไทม์ อัปเดตอัตโนมัติทุก 30 วินาที

**📸 ภาพประกอบ: แดชบอร์ดสถานะห้อง**
```
┌─────────────────────────────────────────────────────────────┐
│  🏨 แดชบอร์ดสถานะห้อง                    🔄 อัปเดต: 10:30 │
│                                                             │
│  สรุป: 🟢 พร้อม: 15  🟡 ทำความสะอาด: 8  🔴 มีผู้พัก: 12  │
│        ⚫ ปิดซ่อม: 2  ⏳ รอตรวจ: 3                         │
│                                                             │
│  กรอง: [ทั้งหมด ▼]  [ว่าง]  [มีผู้พัก]  [ซ่อมบำรุง]      │
│  ค้นหา: [_____________] 🔍                                  │
├─────────────────────────────────────────────────────────────┤
│  ห้อง  │ ประเภท    │ สถานะการพัก │ สถานะทำความสะอาด │ แขก │
├─────────────────────────────────────────────────────────────┤
│  101   │ Deluxe    │ 🟢 ว่าง     │ ✅ ตรวจแล้ว      │  -  │
│  102   │ Deluxe    │ 🔴 มีผู้พัก │ 🧹 สะอาด         │ คุณทดสอบ │
│  103   │ Suite     │ 🟡 ว่าง     │ 🧽 กำลังทำ       │  -  │
│  104   │ Standard  │ 🔴 มีผู้พัก │ 🔴 รอทำ          │ คุณผู้ใช้ │
│  105   │ Deluxe    │ ⚫ ปิดซ่อม   │ 🔧 ซ่อมบำรุง     │  -  │
└─────────────────────────────────────────────────────────────┘
```

### รหัสสีสถานะห้อง

**สถานะการเข้าพัก (Occupancy):**
- 🟢 **ว่าง (Vacant)**: ไม่มีผู้เข้าพัก
- 🔴 **มีผู้พัก (Occupied)**: มีผู้เข้าพักอยู่

**สถานะความสะอาด (Housekeeping):**
- ✅ **ตรวจแล้ว (Inspected)**: พร้อมขายที่สุด
- 🧹 **สะอาด (Clean)**: ทำความสะอาดเสร็จ รอตรวจ
- 🧽 **กำลังทำ (Cleaning)**: กำลังทำความสะอาด
- 🔴 **รอทำ (Dirty)**: รอทำความสะอาด
- 🔧 **ซ่อมบำรุง (Maintenance)**: ต้องซ่อม
- ⚫ **ปิดซ่อม (Out of Service)**: ปิดให้บริการ


### ห้องที่พร้อมสำหรับ Check-in

**ลำดับความสำคัญ:**
1. 🟢 + ✅ (ว่าง + ตรวจแล้ว) - **พร้อมที่สุด**
2. 🟢 + 🧹 (ว่าง + สะอาด) - **พร้อม**
3. 🟢 + 🧽 (ว่าง + กำลังทำ) - **รอสักครู่**
4. 🟢 + 🔴 (ว่าง + รอทำ) - **ยังไม่พร้อม**

### การใช้งานแดชบอร์ด

**การกรอง:**
- คลิกปุ่มกรองเพื่อแสดงเฉพาะห้องที่ต้องการ
- ตัวอย่าง: กรอง "ว่าง" เพื่อดูห้องว่างทั้งหมด

**การค้นหา:**
- พิมพ์หมายเลขห้องหรือชื่อแขก
- ระบบจะกรองผลลัพธ์ทันที

**การรีเฟรช:**
- ระบบรีเฟรชอัตโนมัติทุก 30 วินาที
- คลิกปุ่ม 🔄 เพื่อรีเฟรชด้วยตนเอง

## การทำ Check-in

### เงื่อนไขการ Check-in

✅ **ต้องมี:**
- การจองที่ยืนยันแล้ว
- ห้องว่าง (Vacant)
- ห้องสะอาด (Clean หรือ Inspected)

### ขั้นตอนการ Check-in

**ขั้นตอนที่ 1: ค้นหาการจอง**
1. ไปที่เมนู **"Check-in"**
2. ค้นหาการจองด้วย:
   - หมายเลขการจอง
   - ชื่อแขก
   - เบอร์โทรศัพท์

**📸 ภาพประกอบ: หน้าค้นหาการจอง**
```
┌─────────────────────────────────────────────┐
│  🏨 Check-in                               │
│                                             │
│  ค้นหาการจอง:                              │
│  [หมายเลข/ชื่อ/เบอร์] 🔍                   │
│                                             │
│  📅 แขกที่จะมาถึงวันนี้:                   │
├─────────────────────────────────────────────┤
│  #BK-001234  คุณสมชาย ใจดี                │
│  📅 01/12/2025 - 03/12/2025                │
│  🏨 Deluxe Room (2 คืน)                    │
│  👥 2 คน                                    │
│  [  เริ่ม Check-in  ]                      │
├─────────────────────────────────────────────┤
│  #BK-001235  คุณสมหญิง รักดี               │
│  📅 01/12/2025 - 02/12/2025                │
│  🏨 Suite Room (1 คืน)                     │
│  👥 4 คน                                    │
│  [  เริ่ม Check-in  ]                      │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 2: เลือกห้อง**
1. คลิก **"เริ่ม Check-in"**
2. ระบบจะแสดงห้องที่พร้อม
3. เลือกห้องที่เหมาะสม

**📸 ภาพประกอบ: เลือกห้อง**
```
┌─────────────────────────────────────────────┐
│  เลือกห้องสำหรับ Check-in                 │
│                                             │
│  การจอง: #BK-001234                        │
│  ประเภท: Deluxe Room                       │
│                                             │
│  ห้องที่พร้อม:                             │
├─────────────────────────────────────────────┤
│  ✅ ห้อง 101  (ตรวจแล้ว)  [เลือก]         │
│  ✅ ห้อง 105  (ตรวจแล้ว)  [เลือก]         │
│  🧹 ห้อง 108  (สะอาด)     [เลือก]         │
│  🧽 ห้อง 112  (กำลังทำ)   [เลือก]         │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 3: ยืนยันข้อมูล**
1. ตรวจสอบข้อมูลแขก
2. ตรวจสอบรายละเอียดการจอง
3. แจ้งนโยบายโรงแรม
4. คลิก **"ยืนยัน Check-in"**

**📸 ภาพประกอบ: ยืนยัน Check-in**
```
┌─────────────────────────────────────────────┐
│  ยืนยัน Check-in                           │
│                                             │
│  การจอง: #BK-001234                        │
│  ห้อง: 101 (Deluxe Room)                   │
│                                             │
│  ข้อมูลแขก:                                │
│  👤 คุณทดสอบ ระบบหนึ่ง (หลัก)             │
│  👤 คุณผู้ใช้ ระบบหนึ่ง                    │
│                                             │
│  📅 เข้าพัก: 01/12/2025 - 03/12/2025       │
│  💰 ชำระแล้ว: 5,000 บาท                   │
│                                             │
│  ☑️ ตรวจสอบบัตรประชาชนแล้ว                │
│  ☑️ แจ้งนโยบายโรงแรมแล้ว                   │
│                                             │
│  [  ยืนยัน Check-in  ]  [  ยกเลิก  ]      │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 4: มอบกุญแจ**
1. ระบบจะแสดงข้อมูลกุญแจ
2. มอบกุญแจและแผนที่โรงแรมให้แขก
3. แจ้งเวลา Check-out
4. พิมพ์ใบยืนยัน (ถ้าต้องการ)

**✅ Check-in สำเร็จ**
```
┌─────────────────────────────────────────────┐
│  ✅ Check-in สำเร็จ!                       │
│                                             │
│  ห้อง: 101                                 │
│  กุญแจ: Key Card #101                      │
│  WiFi: HotelWiFi / Password: guest2025     │
│                                             │
│  เวลา Check-out: 12:00 น. (03/12/2025)    │
│                                             │
│  [  พิมพ์ใบยืนยัน  ]  [  เสร็จสิ้น  ]    │
└─────────────────────────────────────────────┘
```

### เคล็ดลับการ Check-in

✅ **ควรทำ:**
- ตรวจสอบบัตรประชาชนทุกครั้ง
- แจ้งนโยบายและเวลา Check-out
- เลือกห้องที่ตรวจแล้วก่อน
- ยิ้มแย้มแจ่มใส ต้อนรับอย่างอบอุ่น

❌ **ไม่ควรทำ:**
- Check-in ห้องที่ยังไม่สะอาด
- ลืมตรวจสอบข้อมูลแขก
- รีบเร่งจนข้อมูลผิดพลาด

## การทำ Check-out

### ขั้นตอนการ Check-out

**ขั้นตอนที่ 1: ค้นหาการจอง**
1. ไปที่เมนู **"Check-out"**
2. ค้นหาด้วยหมายเลขห้องหรือชื่อแขก

**📸 ภาพประกอบ: หน้า Check-out**
```
┌─────────────────────────────────────────────┐
│  🏨 Check-out                              │
│                                             │
│  ค้นหา: [หมายเลขห้อง/ชื่อ] 🔍             │
│                                             │
│  📅 แขกที่จะออกวันนี้:                     │
├─────────────────────────────────────────────┤
│  ห้อง 102  คุณทดสอบ ระบบหนึ่ง             │
│  📅 Check-in: 30/11/2025                   │
│  📅 Check-out: 01/12/2025 (วันนี้)         │
│  [  เริ่ม Check-out  ]                     │
├─────────────────────────────────────────────┤
│  ห้อง 205  คุณผู้ใช้ ตัวอย่างสอง          │
│  📅 Check-in: 29/11/2025                   │
│  📅 Check-out: 01/12/2025 (วันนี้)         │
│  [  เริ่ม Check-out  ]                     │
└─────────────────────────────────────────────┘
```


**ขั้นตอนที่ 2: ตรวจสอบห้อง**
1. ถามแขกว่ามีของหายหรือเสียหายหรือไม่
2. ตรวจสอบมินิบาร์ (ถ้ามี)
3. รับกุญแจคืน

**ขั้นตอนที่ 3: แสดงบิล**
1. ระบบจะแสดงสรุปค่าใช้จ่าย
2. ตรวจสอบกับแขก
3. ยืนยันการชำระเงิน

**📸 ภาพประกอบ: สรุปบิล**
```
┌─────────────────────────────────────────────┐
│  สรุปค่าใช้จ่าย                            │
│                                             │
│  ห้อง: 102 (Deluxe Room)                   │
│  แขก: คุณทดสอบ ระบบหนึ่ง                   │
│  ระยะเวลา: 30/11 - 01/12 (2 คืน)          │
│                                             │
│  รายการ:                                   │
│  ค่าห้อง (2 คืน)         5,000 บาท        │
│  ส่วนลด (SAVE10)          -500 บาท         │
│  ─────────────────────────────             │
│  รวมทั้งสิ้น             4,500 บาท         │
│                                             │
│  ✅ ชำระเงินแล้ว (บัตรเครดิต)             │
│                                             │
│  [  ยืนยัน Check-out  ]  [  พิมพ์บิล  ]  │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 4: ยืนยัน Check-out**
1. คลิก **"ยืนยัน Check-out"**
2. ระบบจะอัปเดตสถานะห้องเป็น "ว่าง + รอทำความสะอาด"
3. แจ้งแม่บ้านให้ทำความสะอาด

**✅ Check-out สำเร็จ**
```
┌─────────────────────────────────────────────┐
│  ✅ Check-out สำเร็จ!                      │
│                                             │
│  ห้อง 102 พร้อมสำหรับทำความสะอาด          │
│  ระบบได้แจ้งแม่บ้านแล้ว                    │
│                                             │
│  ขอบคุณที่ใช้บริการ                        │
│  หวังว่าจะได้ต้อนรับอีกครั้ง              │
│                                             │
│  [  ส่งอีเมลใบเสร็จ  ]  [  เสร็จสิ้น  ]   │
└─────────────────────────────────────────────┘
```

### เคล็ดลับการ Check-out

✅ **ควรทำ:**
- ตรวจสอบห้องก่อน Check-out
- ถามความพึงพอใจของแขก
- แจ้งแม่บ้านทันที
- ส่งอีเมลใบเสร็จ

❌ **ไม่ควรทำ:**
- Check-out โดยไม่ตรวจสอบห้อง
- ลืมรับกุญแจคืน
- ลืมแจ้งแม่บ้าน

## การย้ายห้อง

### เหตุผลในการย้ายห้อง

- 🔧 ห้องมีปัญหา (แอร์เสีย, น้ำไม่ไหล)
- 🔊 แขกร้องเรียนเรื่องเสียงรบกวน
- 🆙 อัพเกรดห้องให้แขก
- 🔄 จัดห้องใหม่ตามคำขอ

### ขั้นตอนการย้ายห้อง

**ขั้นตอนที่ 1: เลือกแขกที่ต้องการย้าย**
1. ไปที่เมนู **"ย้ายห้อง"** หรือ **"Move Room"**
2. ค้นหาห้องปัจจุบัน

**📸 ภาพประกอบ: หน้าย้ายห้อง**
```
┌─────────────────────────────────────────────┐
│  🔄 ย้ายห้อง                               │
│                                             │
│  ค้นหาห้องปัจจุบัน: [102] 🔍              │
│                                             │
│  ข้อมูลห้องปัจจุบัน:                       │
│  ห้อง: 102 (Deluxe Room)                   │
│  แขก: คุณทดสอบ ระบบหนึ่ง                   │
│  Check-in: 01/12/2025                      │
│  Check-out: 03/12/2025                     │
│                                             │
│  เหตุผล: [เลือกเหตุผล ▼]                  │
│  หมายเหตุ: [_________________]             │
│                                             │
│  [  เลือกห้องใหม่  ]                       │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 2: เลือกห้องใหม่**
1. ระบบจะแสดงห้องว่างที่เหมาะสม
2. เลือกห้องใหม่

**📸 ภาพประกอบ: เลือกห้องใหม่**
```
┌─────────────────────────────────────────────┐
│  เลือกห้องใหม่                             │
│                                             │
│  ห้องประเภทเดียวกัน (Deluxe):              │
│  ✅ ห้อง 105  (ตรวจแล้ว)  [เลือก]         │
│  ✅ ห้อง 108  (ตรวจแล้ว)  [เลือก]         │
│                                             │
│  ห้องประเภทสูงกว่า (Suite):                │
│  ✅ ห้อง 201  (ตรวจแล้ว)  [เลือก]         │
│     💰 ไม่มีค่าใช้จ่ายเพิ่ม (อัพเกรด)      │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 3: ยืนยันการย้าย**
1. ตรวจสอบข้อมูล
2. คลิก **"ยืนยันการย้าย"**
3. แจ้งแขกเรื่องห้องใหม่
4. มอบกุญแจใหม่

**✅ ย้ายห้องสำเร็จ**
```
┌─────────────────────────────────────────────┐
│  ✅ ย้ายห้องสำเร็จ!                        │
│                                             │
│  จาก: ห้อง 102                             │
│  ไป:  ห้อง 105                             │
│                                             │
│  กุญแจใหม่: Key Card #105                  │
│                                             │
│  ระบบได้:                                  │
│  ✓ อัปเดตสถานะห้อง 102 เป็น "รอทำ"        │
│  ✓ อัปเดตสถานะห้อง 105 เป็น "มีผู้พัก"    │
│  ✓ แจ้งแม่บ้านทำความสะอาดห้อง 102         │
│                                             │
│  [  เสร็จสิ้น  ]                           │
└─────────────────────────────────────────────┘
```

### เคล็ดลับการย้ายห้อง

✅ **ควรทำ:**
- อธิบายเหตุผลให้แขกเข้าใจ
- เลือกห้องที่ดีกว่าหรือเท่าเทียม
- ช่วยแขกย้ายของ (ถ้าจำเป็น)
- บันทึกเหตุผลในระบบ

❌ **ไม่ควรทำ:**
- ย้ายไปห้องที่ยังไม่สะอาด
- ย้ายโดยไม่แจ้งแขก
- ลืมแจ้งแม่บ้าน

## การจัดการ No-show

### ความหมาย No-show

**No-show** คือ แขกที่จองห้องแต่ไม่มาเช็คอินโดยไม่แจ้งล่วงหน้า

### เงื่อนไข No-show

- เลยเวลา Check-in ปกติ (14:00 น.)
- ไม่ติดต่อโรงแรม
- ไม่ตอบโทรศัพท์

### ขั้นตอนการจัดการ No-show

**ขั้นตอนที่ 1: ตรวจสอบการจอง**
1. ตรวจสอบรายการแขกที่ยังไม่เช็คอิน
2. โทรติดต่อแขก (อย่างน้อย 2 ครั้ง)
3. รอจนถึงเวลาที่กำหนด (เช่น 18:00 น.)

**📸 ภาพประกอบ: รายการ No-show**
```
┌─────────────────────────────────────────────┐
│  🚫 จัดการ No-show                         │
│                                             │
│  แขกที่ยังไม่เช็คอิน (เวลา 18:30):         │
├─────────────────────────────────────────────┤
│  #BK-001234  คุณทดสอบ ระบบหนึ่ง           │
│  📅 Check-in: วันนี้ (14:00)               │
│  📞 โทร: 081-XXX-XXXX                      │
│  🕐 โทรครั้งที่ 1: 15:00 (ไม่รับ)          │
│  🕐 โทรครั้งที่ 2: 17:00 (ไม่รับ)          │
│                                             │
│  [  ทำเครื่องหมาย No-show  ]  [  รอต่อ  ] │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 2: ทำเครื่องหมาย No-show**
1. คลิก **"ทำเครื่องหมาย No-show"**
2. กรอกหมายเหตุ
3. ยืนยัน

**📸 ภาพประกอบ: ยืนยัน No-show**
```
┌─────────────────────────────────────────────┐
│  ⚠️ ยืนยันทำเครื่องหมาย No-show            │
│                                             │
│  การจอง: #BK-001234                        │
│  แขก: คุณทดสอบ ระบบหนึ่ง                   │
│                                             │
│  หมายเหตุ:                                 │
│  [โทรติดต่อ 2 ครั้งแต่ไม่รับสาย]          │
│  [รอจนถึง 18:30 น. แต่ไม่มา]              │
│                                             │
│  ⚠️ การดำเนินการ:                          │
│  • สถานะจะเปลี่ยนเป็น "No-show"            │
│  • ห้องจะกลับมาว่าง                        │
│  • คิดค่าปรับตามนโยบาย                     │
│                                             │
│  [  ยืนยัน  ]  [  ยกเลิก  ]               │
└─────────────────────────────────────────────┘
```

**ขั้นตอนที่ 3: ปล่อยห้อง**
- ระบบจะปล่อยห้องให้ว่าง
- สามารถขายห้องให้แขกคนอื่นได้
- บันทึกประวัติ No-show

### นโยบาย No-show

- คิดค่าปรับ 1 คืน (ตามนโยบาย)
- ไม่คืนเงินมัดจำ
- บันทึกประวัติเพื่อป้องกันในอนาคต

## การจัดการการจอง

### การสร้างการจองใหม่

พนักงานต้อนรับสามารถสร้างการจองให้แขกได้โดยตรง

**ขั้นตอน:**
1. ไปที่เมนู **"สร้างการจอง"**
2. กรอกข้อมูลแขก
3. เลือกห้องและวันที่
4. ยืนยันการจอง

### การแก้ไขการจอง

**สิ่งที่แก้ไขได้:**
- ข้อมูลแขก (ชื่อ, เบอร์โทร)
- จำนวนผู้เข้าพัก
- หมายเหตุพิเศษ

**สิ่งที่แก้ไขไม่ได้:**
- วันที่เข้าพัก (ต้องยกเลิกและจองใหม่)
- ประเภทห้อง (ต้องยกเลิกและจองใหม่)

### การยกเลิกการจอง

พนักงานต้อนรับสามารถยกเลิกการจองได้ตามนโยบาย

**ขั้นตอน:**
1. ค้นหาการจอง
2. คลิก **"ยกเลิก"**
3. เลือกเหตุผล
4. ยืนยันการยกเลิก

## คำถามที่พบบ่อย (FAQ)

### การ Check-in

**Q: แขกมาถึงก่อนเวลา Check-in ทำอย่างไร?**  
A: ตรวจสอบว่ามีห้องพร้อมหรือไม่ ถ้ามีสามารถ Early Check-in ได้ (อาจมีค่าใช้จ่ายเพิ่ม)

**Q: แขกไม่มีบัตรประชาชน?**  
A: ขอเอกสารอื่นที่มีรูปถ่าย เช่น ใบขับขี่, พาสปอร์ต

**Q: ห้องที่จองไม่พร้อม?**  
A: เสนอห้องอื่นประเภทเดียวกัน หรืออัพเกรดฟรี

### การ Check-out

**Q: แขกต้องการ Late Check-out?**  
A: ตรวจสอบว่ามีการจองห้องนั้นในวันเดียวกันหรือไม่ ถ้าไม่มีสามารถอนุญาตได้ (อาจมีค่าใช้จ่ายเพิ่ม)

**Q: แขกทำของหาย/เสียหาย?**  
A: บันทึกรายละเอียด ถ่ายรูป และแจ้งผู้จัดการ

**Q: แขกไม่พอใจบริการ?**  
A: รับฟังปัญหา ขอโทษ และเสนอวิธีแก้ไข (ส่วนลด, อัพเกรดครั้งหน้า)

### ระบบ

**Q: ระบบช้าหรือค้าง?**  
A: รีเฟรชหน้าเว็บ หรือแจ้งฝ่าย IT

**Q: ข้อมูลไม่ตรงกับความเป็นจริง?**  
A: รีเฟรชแดชบอร์ด หรือตรวจสอบกับแม่บ้าน

**Q: ไม่สามารถ Check-in ได้?**  
A: ตรวจสอบว่าห้องว่างและสะอาดหรือไม่ ถ้ายังไม่สะอาดให้รอหรือเลือกห้องอื่น
